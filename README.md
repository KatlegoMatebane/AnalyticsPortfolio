<h3><b>About Repository</b></h3>
This repository serves as a data analyst portfolio built around a fictional company, Cacao & Co., and showcases a collection of projects using a single, consistent dataset sourced from Kaggle. Each project is designed to simulate real-world scenarios I’ve worked on across the Banking and iGaming industries, demonstrating practical, end-to-end analytical thinking.

The portfolio includes a range of data-focused solutions such as extracting structured data from ID documents, developing an interactive Power BI dashboard, producing an annual performance report, and building a competition simulator. It also features analytical frameworks for a VIP programme and an affiliate programme, reflecting common business use cases in customer segmentation, performance tracking, and revenue optimization.

All projects in this repository are developed using tools commonly applied in industry, including Python, SQL, VS Code, Adobe InDesign, and Adobe Photoshop. By working with one dataset across multiple projects, the repository highlights how different analytical approaches, tools, and business problems can be applied to the same data—mirroring how real-world data analysts continuously derive new insights from existing data sources.
